# API - Health Check

## Objetivo

Validar a disponibilidade básica da API e confirmar que o serviço está ativo.

---

## API-001 - Health Check

| Campo | Valor |
|---------|---------|
| Método | GET |
| Endpoint | /health |
| Objetivo | Verificar se a API está disponível e respondendo corretamente |
| Resultado Esperado | Status 200 e resposta indicando que o serviço está ativo |
